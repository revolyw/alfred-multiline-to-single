# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security issue in this project, please **do not** open a
public GitHub issue.

Preferred options:

1. Use [GitHub Private Vulnerability Reporting](https://github.com/revolyw/alfred-multiline-to-single/security/advisories/new) (if enabled on the repository)
2. Contact the maintainer via a private channel linked from the GitHub profile [@revolyw](https://github.com/revolyw)

Please include:

- A description of the issue and potential impact
- Steps to reproduce or a proof of concept
- Affected version / commit if known

We will acknowledge reports as soon as practical and work on a fix. Please allow
reasonable time before any public disclosure.
